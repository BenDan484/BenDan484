# 三月复盘

三月复盘的核心思路可以浓缩为**自我成长、反内耗、务实行动、人际边界**四大主线：



1. **学习与生活实践**

   完成心理学书籍阅读；科目三考试虽波折但最终通过，暴露临场心态与流程熟练度问题。

2. **人际关系与情绪边界**

   确立 “只对值得的人付出情绪” 的原则；以**边界感**避免内耗，不被他人消耗、不纠结不确定关系。

3. **反内耗与思维升级**

   停止反刍思维，不恋过往、不忧未来；用辩证思维看问题，接受自身局限与坏事发生，专注当下该做的事。

4. **金钱与行动认知**

   把钱当作解决问题的工具，而非目的；认清自身水平，聚焦关键任务，借力他人与资源提升效率，放下面子求成长。

5. **长期成长心态**

   接纳 “艰难是常态”，用每日小进步积累质变；保持耐心与高认知，专注自我提升，活在真实感受里。

银行贷款不是把现金贷出去,而是通过准备金凭空制造出新的存款?

这句话非常准，而且是现代银行体系最反直觉的真相：
银行放贷，不是把别人的存款“借出去”，而是在放贷那一刻，直接创造了新的存款（新钱）。

我用最通俗、不绕弯的方式给你讲透。

一、你最关心的结论

银行不是“中介”：
不是收了100块存款，再贷出90块。
而是：
你申请贷款 → 银行批准 → 银行在你账户上敲个数字** → 新钱就诞生了。**

贷款创造存款，不是存款创造贷款。

二、极简过程（一看就懂）

  1. 你去银行贷 100万买房
  2. 银行不需要先有这100万现金
  3. 银行直接在你账户里写：存款 +100万
  4. 同时在自己账上记：贷款资产 +100万

就这么简单：一笔贷款 = 一笔新存款。

你账户里多出来的100万，之前根本不存在。

三、那“存款准备金”是干嘛的？

准备金不是用来“准备放贷”的，
而是用来应对提现和转账的。

  • 银行不能无限创造货币
  • 央行会要求：银行必须留一点点钱（准备金），保证有人取钱、转账时能付出去
  • 准备金限制规模，但不创造钱

逻辑是：
先有贷款 → 才有存款 → 再按比例留准备金
不是：
先有存款 → 扣掉准备金 → 剩下才能贷。

四、和你之前问题的关系：通胀从哪来？

你问：买房抗通胀。
通胀的一大来源，就是：
银行大量放贷 = 大量创造新存款 = 钱变多 = 钱贬值

所以你直觉完全没错：

  • 现金会被信贷扩张稀释
  • 房子是对抗这种货币被凭空创造出来的硬资产之一

五、一句话终极总结

银行不是把现金贷出去，
而是在你贷款获批的那一刻，
直接在账户上创造出新的存款。
贷款，才是现代货币的真正来源。
